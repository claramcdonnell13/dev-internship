Design Questions

  1. Design is not about appearance. Design is about the entire experience of using a site. This does include appearance, but it also includes vastly important things like functionality, usability, and the overall user experience. I believe that designers and developers should work closely together in developing how a site functions overall, and how usability plays into the design.
  2. Grids are useful for creating a guide for how to present content in an organized, uniform manner. They design based on proportionality, which can be beneficial when creating for a variety of platforms. I begin a project with a grid system, using it to organize content from the beginning of a project.
  3. Pattern libraries include standards and code for different elements of design, such as buttons or nav bars. These libraries can be useful for standardization across a variety of products and can add ease to the design process.
